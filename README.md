API REST de Lista de Tarefas (To-Do List) Este projeto consiste em uma API REST desenvolvida em Node.js + Express, utilizando Sequelize ORM com banco de dados SQLite.

A API permite gerenciar uma lista de tarefas, possibilitando:
- Criar tarefas
- Listar todas as tarefas
- Buscar tarefa por ID
- Atualizar uma tarefa
- Atualizar apenas o status
- Excluir tarefas
  
Como Rodar o Projeto:
- Clonar o repositório
- Instalar dependências: npm install
- Iniciar o servidor: node server.js
- Servidor rodando: http://localhost:3000
- Abra no navegador: http://localhost:3000/tarefas
